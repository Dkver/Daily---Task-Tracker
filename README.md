
# Task Tracker - Frontend Project

Task Tracker is a task management application built to demonstrate my ability to design and develop structured, interactive frontends using modern technologies like React and TypeScript. The app allows users to create, organize, and manage tasks efficiently with an intuitive and responsive user interface.

I implemented features such as dynamic task creation, updating, and deletion, along with smooth state management to ensure a seamless user experience. Data persistence is handled using localStorage, keeping the application lightweight and fully functional without a backend.

The project focuses on clean code practices, modular architecture, and scalability, making it easy to extend with additional features in the future.
